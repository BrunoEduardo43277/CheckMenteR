# CheckMente

Plataforma web inteligente desenvolvida para monitoramento emocional, acolhimento digital e prevenção de riscos emocionais no ambiente escolar.

# Sobre o projeto

O CheckMente é uma plataforma educacional criada com o objetivo de auxiliar escolas no acompanhamento emocional de estudantes através de tecnologia, inteligência artificial e análise de dados emocionais.

A plataforma permite:

- Check-ins emocionais
- IA de acolhimento
- Alertas preventivos
- Relatórios emocionais
- Mapa emocional inteligente
- Comunicação escolar
- Monitoramento de padrões emocionais

# Objetivo

Promover acolhimento emocional, prevenção de crises, inclusão digital e apoio escolar utilizando tecnologia moderna e acessível.
